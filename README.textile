h1. Just Another Blog Engine

h3. What is this?

JABE is a bare bones blogging engine that is installed as a gem. It will grow as its needs do.

This version is for Rails 3.1+

h2. Installing

Edit your Gemfile and add the required gems

<pre><code>gem 'devise'
gem 'devise', '>= 1.5.3'
gem 'friendly_id', '>= 4.0.0'
gem 'haml-rails', '>= 0.3.4'
gem 'kaminari'
gem 'simple_form', '>= 1.5.2'
</code></pre>

From your rails root

<pre><code>bundle install
rails plugin install http://github.com/defunkt/acts_as_textiled.git
rails g devise:install
</code></pre>

Edit your Gemfile and add JABE

<pre><code>gem 'jabe'
</code></pre>

From your rails root

<pre><code>rails g jabe:migrations
rake db:migrate
rm app/views/layouts/*
rm public/index.html

rails runner "Admin.create!(
  :email => 'you@example.com',
  :password => 'password',
  :password_confirmation => 'password'
)"

rails s
http://localhost:3000/admin
edit settings
</code></pre>

h2. Upgrading from 0.5.*

<pre><code>rails g migration rename_cached_slug_on_entries
</code></pre>

Edit the migration to look like

<pre><code>class RenameCachedSlugOnEntries < ActiveRecord::Migration
  def change
    rename_column :entries, :cached_slug, :slug
  end
end
</code></pre>
